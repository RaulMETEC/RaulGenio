BANCO DE DADOS

Professores: Iury da Silva
Dias de aula: Quinta-Feira

Visão Geral
A matéria de Banco de Dados é uma matéria tecnica que tem como objetivo o ensinamento sobre como fazer banco de dados com fluxogramas ER(entidade e relacionamento) também falando sobre alguns aplicativos como o "Dia".

Opinião Pessoal:
A matéria de Banco de Dados é divertida, o professor normalmente passa uma prova no bimestre, porém as provas não são difíceis se você prestar atenção nas aulas.
